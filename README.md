# StreakBuilt PWA — Install Guide

## How to install on your phone (5 minutes)

### Step 1 — Create a free GitHub account
Go to https://github.com and sign up (free).

### Step 2 — Create a new repository
- Click the **+** icon → **New repository**
- Name it: `streakbuilt`
- Set it to **Public**
- Click **Create repository**

### Step 3 — Upload these files
- Click **uploading an existing file**
- Drag ALL 4 files into the box:
  - `index.html`
  - `manifest.json`
  - `sw.js`
  - `icon-192.png`
  - `icon-512.png`
- Click **Commit changes**

### Step 4 — Enable GitHub Pages
- Go to **Settings** tab
- Click **Pages** in the left sidebar
- Under "Source" select **main** branch → **/ (root)**
- Click **Save**
- Wait ~1 minute

### Step 5 — Open your app URL
Your app will be live at:
`https://YOUR-USERNAME.github.io/streakbuilt`

Open that URL in Chrome on your phone → you'll see an **INSTALL** banner appear → tap it → it's on your home screen like a real app! ✓
